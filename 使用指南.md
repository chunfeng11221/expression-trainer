# 表达力训练器 · 使用指南

这份指南带你完整走一遍训练流程,并说明每个页面怎么看、每个按钮有什么用。
还没装好的话,先按 README 的三步来:双击「一键安装.bat」→ 双击「启动训练器.bat」→ 首页粘贴 API Key。

---

## 一、产品在练什么

练的是**一分钟口头表达**。核心闭环:

```
一道题 → 准备 15 秒 → 说 1 分钟 → AI 分析 → 具体反馈 → 同一题再说一次 → 看到进步
```

理念:先说出来,再说清楚。不需要完美,只需要再说一次。

---

## 二、开始一次训练

### 第 1 步:首页

打开 http://127.0.0.1:8788 看到首页,有两条路:

- **「随机开始」**:系统随机抽一道题,直接进准备倒计时。不想选题就点它
- **「选择题目」**:自己挑一道题(见下)

副标题下面有一行小字是今日牵引:练过了显示「今天已练 2 次 · 连续 3 天」,今天还没练显示「今天还没开口,来一题?」——只是信息,不打卡不排名。

首页下方的小字显示当前训练模式(默认「1分钟表达 · 15秒准备 · 汇报 · 普通观众」),旁边的「修改」可以改这些参数;「训练历史」「进步表格」入口也在这一行附近。

> 第一次使用、还没配 AI 时,首页会出现「接入 AI」卡片:默认已选好 DeepSeek,卡片上会写明去哪个网站注册拿 key,粘贴过来点「保存并测试」即可。不配也能练,只是反馈没那么细致。

### 第 2 步:选题(走了「选择题目」的话)

顶部五个分类,按你想练的场景选:

| 分类 | 练什么 | 例子 |
| --- | --- | --- |
| 日常 | 轻松地聊一件事 | 你最近改变过的一个习惯是什么? |
| 观点 | 清晰表达立场 | 努力一定会带来回报吗? |
| 工作 | 汇报、沟通、拒绝 | 向团队汇报一个项目延期 |
| 解释 | 把复杂的事讲明白 | 向小学生解释什么是通货膨胀 |
| 公考面试 | 公务员结构化面试(社会现象/态度观点/计划组织/应急应变/人际关系/情景模拟) | 现在人工智能发展迅速,你怎么看? |

卡片上的「简单 / 普通 / 困难」是难度参考;公考面试题的卡片还会带一个子题型小标签(如「社会现象」)。新手建议从「日常」或「工作」的简单题开始。

> 选中公考面试题时,自动切换到面试节奏:**思考 1 分钟、作答 3 分钟**(准备页显示「思考时间」,录音页显示「作答时间」)。你在设置里手动改过回答/准备时间后,就按你的设置来。

### 第 3 步:准备 15 秒

屏幕显示:**题目 + 倒计时 + 思考提示 + 「立即开始」按钮**。

- 思考提示是 AI 针对这道题生成的(没配 AI 时是通用四条),只给思考角度,不给答案——答案得是你自己的
- 建议用法:花 5 秒定一个核心观点,再花 10 秒想两点理由和一个例子。**不要打逐字稿**,写几个关键词就够
- 倒计时下面有一行小字提醒:到点会自动开始录音。想好就点「立即开始」,不用等满 15 秒

### 第 4 步:录音 1 分钟

第一次录音浏览器会请求麦克风权限,点「允许」。(不小心拒绝了:点地址栏左侧的小锁图标,把麦克风改成允许,再点页面的重试。)

录音页面上有什么:

- **已用时间 / 剩余倒计时**:默认 1 分钟,到点自动结束
- **音量条**:说话时跳动,确认麦克风正常工作
- **实时字幕**(Chrome/Edge):说的话实时变文字;高频口癖(然后、就是、那个……)会用**波浪线**标出来,角落有轻量统计如「然后 × 4」——只是让你看到,不会打断你
- **「结束录音」**:说完了随时点,不用撑满
- **「重新开始」**:这次说砸了,点它重录本题。它会弹一次确认(按钮就在「结束录音」旁边,防止手滑毁掉一段好录音)

> 停顿不是错误。想两秒再说,比用「然后」填满两秒好得多。

### 第 5 步:转写与分析(等待页)

录音结束后系统在做两件事:

1. **转写**:本地 Whisper 模型把你的录音转成文字(1 分钟录音约几十秒;面试那种 3–5 分钟的长录音可能要等 1–2 分钟,页面会提前告诉你,别刷新)
2. **AI 分析**:DeepSeek 等 AI 阅读文字稿,从观点、结构、内容、流畅度四个维度评价(约 5-10 秒;没配 AI 时由本地规则分析,同样可用)

什么都不用点,等它自己跳结果页。

### 第 6 步:读懂结果页

从上到下:

- **一句话诊断**:这次表达最关键的一个优点 + 一个问题,先读它
- **四项评分**(各 100 分):观点 / 结构 / 内容 / 流畅度。分数下面有一行小字,说明这道题里该维度看什么(比如介绍类题目的「观点」看的是"核心信息是否讲清",不是"立场")
- **最好的两点**:你做得对的地方,下次继续保持
- **最需要修改的两点**:每次只给你两个主要问题——别贪多,就改这两个。这个区块后面就有一个主按钮(「根据建议再说一次」/随心记是「再记一段」),不用滚到页面底部找
- **基础数据**:时长、总字数、语速(字/分钟,中文汇报参考区间 180–260)、口癖总数、最长停顿、观点第几秒出现、有没有结尾
- **完整文字稿**:按段落带时间(如 00:00–00:12),口癖有波浪线,句子带标签(观点/理由/例子/总结);有录音可回放时,**点任意一段会从该段的时间继续播放**——听到哪句不对,点那句就行(刷新页面后录音不在内存里,段落就不可点了)
- **具体修改建议**:每条都引用你原话,告诉你那句话可以怎么改;口癖多的时候,建议会直接给替代说法(比如「然后」→「接下来/与此同时」)
- **改进提纲**:基于你原本的观点,给出第二次回答的 5 步结构——不是标准答案,是你的内容的更好排列

### 第 7 步:根据建议再说一次(关键)

点结果页底部最醒目的**「根据建议再说一次」**(页面中间「最需要修改的两点」后面也有同一个按钮):

1. 页面先给你本次最重要的**两个改进目标**(就盯这两个);到了录音页,这两个目标还会以一行小字挂在顶部,说着说着忘了可以瞟一眼
2. 再准备 15 秒 → 重新录 1 分钟 → 再次分析
3. 自动进入**对比页**:两次的各项分数变化、口癖变化、观点出现时间变化、有没有新增例子和结尾,顶部一句话总结"这一次具体进步在哪里"

同一个题目说两遍,是这个产品最核心的训练动作。

## 三、训练历史

首页「训练历史」进入。每一次练习(包括重练)都会自动存档:

- **按训练分组**:同一题的「第一次 → 重练」排在一起,带分数升降箭头
- **分类筛选**:记录多了以后,顶部可以按题型筛(全部/公考面试/日常/观点/工作/解释/随心记),只出现你练过的分类
- 点任意一条进**详情页**:完整分析 + 文字稿 + **回放当时的录音**(文字稿段落同样可以点击跳播)
- 一组里有两次练习时,可以点「查看对比」
- 每条可删除(会二次确认);总量超过 100 条自动淘汰最旧的

## 四、进步表格

首页「进步表格」进入,看长期趋势:

- **顶部摘要**:总练习次数、近 5 次平均分 vs 前 5 次、最佳成绩、口癖变化
- **五维折线图**:综合 + 观点/结构/内容/流畅度五条线,**点图例可以显隐某条线**;鼠标悬停看每一点的日期和分数
- **明细表**:每次练习一行,与上一次比较的升降箭头。注意口癖和"观点出现时间"是**反向指标**——降了才是进步

## 五、训练设置(首页「修改」)

| 参数 | 选项 | 建议 |
| --- | --- | --- |
| 回答时间 | 30秒 / 1分钟 / 2分钟 / 3分钟 / 4分钟 / 5分钟 | 新手 1 分钟;练熟了用 2 分钟练完整汇报;公考面试作答约 3 分钟 |
| 准备时间 | 0秒 / 15秒 / 30秒 / 1分钟 | 默认 15 秒;0 秒模拟即兴发言;公考面试思考约 1 分钟 |
| 场景 | 汇报 / 即兴 / 面试 / 公考面试 | AI 会按场景调整评价口径 |
| 受众 | 普通观众 / 领导 / 面试官 / 专业人士 | 同上 |

保存即生效,首页小字会同步。注意:手动改过回答/准备时间后,公考面试题不再自动套用面试节奏,以你的设置为准。

## 六、怎么练效果好

- **少量多次**:每天 2–3 题,比周末一次练 20 题有用
- **同一题说 2–3 遍**:第一遍发现问题,第二遍改,第三遍巩固;三遍的分数曲线就是最好的反馈
- **一次只改一个毛病**:比如这周专盯"口癖",下周专盯"观点前置"
- **回听自己的录音**:文字稿告诉你说错了什么,录音告诉你"听起来"怎么样
- **换受众再说一遍**:同一道题,对"普通观众"和对"领导"说,是两练

## 七、使用中可能遇到的情况

- **分析结果角落写「本地分析」**:说明这次没用上 AI(key 没配 / 欠费 / 网络问题),反馈依然可用,只是没那么细致;去设置页检查 AI 配置
- **转写失败或文字稿明显不对**:先确认麦克风没静音、说话音量正常;重录一次即可。偶尔模型没下完会自动补下,等一两分钟再试
- **录音没声音/音量条不动**:检查系统麦克风输入设备是否选对,浏览器标签页是否被静音
- **想清空所有数据**:浏览器里开 http://127.0.0.1:8788,按 F12 → 控制台输入 `localStorage.clear()` 回车,再输入 `indexedDB.deleteDatabase('expression-trainer')` 回车,刷新即可

更多安装/启动类问题见 README 的常见问题。

## 八、隐私

- 录音、文字稿、分析、历史全部存在**你自己电脑**上(localStorage + IndexedDB),不上传任何服务器
- API Key 保存在本机 `server/ai.config.json`,不会进 Git、不会外发
- 唯一的网络请求:转文字稿发给 AI 做分析(你配置的服务商);不配 AI 则完全不联网也能用
