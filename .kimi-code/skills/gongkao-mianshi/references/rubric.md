# 公考面试四维评分标准(分数锚点)

四个维度名称固定:观点 / 结构 / 内容 / 流畅度,均为 100 分制。评分严格按锚点校准,不和稀泥——内容空洞就该 60 以下,优秀才上 90。

## 观点(审题与立场)

- 90+:完全切题,首段即亮明态度/处理原则,立场鲜明且全文一致,能透过现象看到本质
- 75–89:切题,有明确态度但出现稍晚,或表态有了但立意深度一般
- 60–74:基本围绕题目但偶有游离;态度含糊("辩证看待"当挡箭牌,实际没有立场)
- <60:跑题偏题,或通篇没有自己的态度

**硬性扣分**:"怎么看"类题首段不表态 → ≤65。

## 结构(逻辑条理)

- 90+:框架完整(表态→分析→对策→收尾 或 是什么—为什么—怎么办),分点清晰(第一/第二/第三 或 首先/其次/再次/最后),每段末尾扣题,详略得当
- 75–89:有框架有分点,但个别段落游离或收尾仓促/缺失
- 60–74:有大概先后,但分点不清、层次粘连,听不出"几点"
- <60:想到哪说到哪,无明显结构

**硬性扣分**:全程不分点 → ≤65。

## 内容(分析与对策)

- 90+:分析有层次(主体分析:个人/企业/政府/社会;或层面分析:思想/制度/监管),有事例或热点积累,对策有主体有抓手("谁,做什么,怎么做"),辩证克制
- 75–89:有分析有对策但深度一般,或对策可行但缺少事例支撑
- 60–74:分析和策论都有但偏笼统("加强监管""提高意识"式正确废话居多)
- <60:通篇空喊口号、重复题干、以情绪代替分析

**硬性扣分**:对策只有口号 → ≤65;极端化表述(一竿子打翻一船人、以偏概全)→ ≤70。

## 流畅度(言语表达)

- 90+:像当面说话,节奏自然;口头稿口癖≤2 次,语速 180–260 字/分钟,无长时间卡壳
- 75–89:整体流畅,口癖 3–5 次或个别长停顿
- 60–74:口癖 6–9 次,或明显背稿腔、频繁重起句子
- <60:口癖≥10 次、卡壳严重、读稿感强

**硬性扣分**:口癖≥8 次 → ≤70;严重超时或不足一半时长 → ≤65。

## 综合分

综合分 = 四项加权(观点 30% / 结构 25% / 内容 30% / 流畅度 15%),与单项校准一致,不另起炉灶。
